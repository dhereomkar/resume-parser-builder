# Resume Parser & Resume Builder

A full-stack web application that allows users to upload PDF resumes, automatically extract key information, edit the parsed results, and generate professional resumes using customizable templates.

This project is developed as an internship task and demonstrates practical usage of Node.js, Express, and frontend web technologies.
